# Simple HTML & CSS Layout

Este repositório contém uma página HTML simples com estilização em CSS, criada para praticar estruturação de conteúdo, pseudo‑elementos e manipulação visual de texto.

## 📌 Sobre o projeto
A página demonstra:
- Uso de listas com pseudo‑elementos (`::after`)
- Estilização tipográfica avançada com `::first-letter` e `::first-line`
- Personalização de seleção de texto (`::selection`)
- Estrutura HTML limpa e sem dependências externas

Este projeto faz parte do meu percurso de estudos em **HTML, CSS e desenvolvimento web**, onde pratico conceitos fundamentais para criar interfaces modernas e bem estruturadas.

## 🧩 Tecnologias utilizadas
- **HTML5**
- **CSS3**
