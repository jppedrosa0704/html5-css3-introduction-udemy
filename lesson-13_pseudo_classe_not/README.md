📄 README — Projeto HTML & CSS (Pseudo‑classes e Seletores)
Este repositório contém um projeto simples em HTML5 e CSS3, criado para praticar pseudo‑classes, seletores avançados, estilização de elementos e comportamento dinâmico com :hover, :focus, :not, :checked, :required, entre outros.

🚀 Objetivo do Projeto
O foco deste exercício é demonstrar:

Uso de pseudo‑classes CSS

Seletores avançados como :not(), :first-child, :required, :focus

Comportamento visual dinâmico em links, inputs e listas

Estruturação básica de HTML com listas, inputs, textos e divs

Organização de estilos externos via arquivo styles.css

🧩 Estrutura do Projeto
Código
/
│── index.html
│── assets/
│     └── css/
│           └── styles.css
🎨 Destaques do CSS
Links estilizados com :link, :visited, :hover, :active

Inputs com foco usando input:focus

Campos obrigatórios estilizados com input:required

Seletores condicionais como input:not(:checked) + p

Pseudo‑classe estrutural li:first-child

Uso de :not() para estilizar todos os elementos exceto um específico

🖥️ Demonstração dos Elementos
O projeto inclui:

Inputs de texto

Checkboxes

Radio buttons

Parágrafos com links

Lista dentro de container .pai

Div com seletor :not() aplicado

Estilos de fundo, cores e transições

HTML5

CSS3
