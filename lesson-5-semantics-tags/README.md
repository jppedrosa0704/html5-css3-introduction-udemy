🧩 HTML Layout Structure – Practice Project
This repository contains a simple HTML document created to practice semantic structure, layout organization, and the use of common HTML5 elements such as <header>, <section>, <article>, <aside>, and <nav>.

The project focuses on building a clean and readable page structure without CSS, emphasizing semantic meaning and proper content grouping.

📄 Project Overview
The HTML file demonstrates:

Semantic layout using:

<header> for introductory content

<section> to group related topics

<article> for independent content blocks

<aside> for secondary navigation

<nav> for menu structure

Proper use of headings (<h1> to <h3>)

Clean and organized indentation

Readable and accessible HTML structure

🧱 Page Structure
The page is divided into:

Main Header  
Contains the site title and a short introductory paragraph.

Sidebar Navigation  
A simple list of links inside an <aside> element.

Section: Roupas Femininas  
Includes three articles with headings and placeholder text.

Section: Roupas Masculinas  
Another set of articles demonstrating repeated structure.

🎯 Purpose of This Project
This project was created to practice:

Semantic HTML5 tags

Structuring content logically

Improving readability and maintainability

Preparing for future styling with CSS

It serves as a foundation for more advanced layouts and styling exercises.
