📘 Basic HTML Page Example
This repository contains a simple HTML page created to practice the fundamentals of HTML5 and CSS.
The project demonstrates how to structure a webpage, apply basic styling, and work with IDs, classes, and inline CSS.

📄 Project Overview
The HTML file includes:

Proper document declaration using <!DOCTYPE html>

Language attribute set to Brazilian Portuguese (lang="pt-BR")

UTF‑8 character encoding

Responsive viewport configuration

Inline CSS demonstrating:

Background colors

Class selectors

ID selectors

Text styling with color and bold emphasis

🎨 Features Demonstrated
Headings (<h1>) with different backgrounds

Paragraph (<p>) with styled text

Classes such as gray-background, bold, red-background, red-border

IDs such as heading-one and heading-two

Use of RGBA colors for transparency effects

🚀 How to View
Simply open the HTML file in any modern web browser to see the rendered page.

📚 Purpose
This exercise is part of my learning path in HTML5 and CSS3, focusing on:

Understanding document structure

Practicing selectors (ID, class, element)

Applying basic styling

Improving code organization and readability
