📘 Basic HTML Structure and Styling
This repository contains a simple HTML page created to practice the fundamentals of HTML5 and basic CSS styling.
The project demonstrates how to structure a webpage, apply inline and internal styles, and work with headings, paragraphs, and custom classes.

📄 Project Overview
The HTML file includes:

Proper document declaration using <!DOCTYPE html>

Language attribute set to Brazilian Portuguese (lang="pt-BR")

UTF‑8 character encoding

Responsive viewport configuration

Internal CSS with a custom class (.blue-color)

Inline styling applied to paragraphs

Demonstration of all heading levels (<h1> to <h6>)

🎨 Features Demonstrated
Headings (<h1> to <h6>) to show hierarchy and typography

Paragraphs (<p>) with inline font-size styling

Class selector (.blue-color) to apply text color

Use of Lorem ipsum placeholder text

Clean and readable HTML structure

🚀 How to View
Open the HTML file in any modern web browser to see the rendered page.

🎯 Purpose
This exercise is part of my HTML5 learning path, focusing on:

Understanding document structure

Practicing heading hierarchy

Applying inline and internal CSS

Improving code readability and organization
