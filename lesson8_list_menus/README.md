📄 Projeto HTML – Barra de Navegação e Listas
Este repositório contém uma página HTML simples desenvolvida para praticar conceitos fundamentais de front‑end, incluindo criação de listas, barra de navegação e estilização básica com CSS. O objetivo é reforçar a estrutura semântica e o uso correto de elementos essenciais da linguagem.

🎯 Objetivo do Projeto
Este projeto foi criado para treinar:

Estruturas de listas HTML:

<ul> (lista não ordenada)

<ol> (lista ordenada)

<dl> (lista de definição)

Personalização de listas com atributos (type="I", type="A")

Construção de uma barra de navegação com <nav> e Flexbox

Conceitos básicos de CSS:

Reset de margens e padding

Alinhamento com Flexbox

Efeitos de hover

Tipografia e espaçamento

🧱 Funcionalidades
✔️ Barra de Navegação
Criada com:

Estrutura semântica <nav>

Alinhamento central com Flexbox

Links estilizados e com efeito hover

Texto em maiúsculas e fonte personalizada

✔️ Listas Não Ordenadas
Demonstração simples de <ul> com marcadores padrão.

✔️ Listas Ordenadas
Exemplos com diferentes estilos:

Numeração padrão

Algarismos romanos (type="I")

Letras maiúsculas (type="A")

✔️ Lista de Definição
Uso de <dl>, <dt> e <dd> para representar termos e descrições.

This repository contains a simple HTML page created to practice essential front‑end concepts, including list structures, a navigation bar, and basic CSS styling. The project focuses on semantic HTML and clean, organized code.

🎯 Project Purpose
This project was built to practice:

HTML list structures:

<ul> (unordered list)

<ol> (ordered list)

<dl> (definition list)

Custom list formatting (type="I", type="A")

Building a navigation bar using <nav> and Flexbox

Basic CSS concepts:

Margin and padding reset

Flexbox alignment

Hover effects

Typography and spacing

🧱 Features
✔️ Navigation Bar
Built using:

Semantic <nav> element

Centered layout with Flexbox

Styled links with hover effect

Uppercase text and custom font

✔️ Unordered Lists
A simple <ul> demonstrating default bullet styling.

✔️ Ordered Lists
Examples using different numbering formats:

Default numeric

Roman numerals (type="I")

Alphabetical (type="A")
