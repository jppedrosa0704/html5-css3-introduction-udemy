📄 README.md — Projeto HTML com Background e Header
🖥️ Sobre o Projeto
Este repositório contém uma página HTML simples criada para treinar estruturação de layout, aplicação de estilos com CSS e uso de imagens de fundo responsivas.
O objetivo principal é demonstrar:

Uso de background-image com cover

Criação de um header fixo e transparente

Organização semântica básica em HTML

Estilização com CSS puro

🎨 Funcionalidades Implementadas
Imagem de fundo em tela cheia  
Utilizando background-size: cover e background-position: top center.

Header com transparência  
Aplicado com rgba(0,0,0,0.5) para criar contraste com o fundo.

Links estilizados  
Navegação simples com links “Login” e “Logout”.

Reset básico de CSS  
Remoção de margens e paddings padrão.

🧩 Estrutura do Código
HTML
Uso de tags semânticas (header, div, h1, h2)

Links de navegação

Estrutura simples e limpa

CSS
Reset global com *

Background configurado diretamente no body

Flexbox para alinhar os itens do header

Tipografia com sans-serif

🖼️ Imagem de Fundo
