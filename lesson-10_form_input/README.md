📄 Formulário HTML5 – Inputs e Elementos de Formulário
Projeto desenvolvido como parte do curso HTML5 & CSS3 – Udemy, demonstrando diversos tipos de inputs, boas práticas de estruturação e uso de estilos básicos em CSS.

🔗 Página publicada:  
https://jppedrosa0704.github.io/html5-css3-introduction-udemy/lesson-10_form_input/

📌 Sobre o Projeto
Este projeto apresenta um formulário completo utilizando elementos modernos do HTML5.
O objetivo é demonstrar, de forma prática, como diferentes tipos de campos funcionam e como podem ser aplicados em formulários reais.

Inclui exemplos de:

Campos de texto

Checkboxes

Radio buttons

Inputs de data, hora e datetime-local

Seletor de cor

Campo de email

Campo de password

Upload de ficheiros

Input numérico

Slider com datalist

Campo de pesquisa

Select com optgroups

Textarea

Botões de reset e submit

🧩 Tecnologias Utilizadas
HTML5

CSS3 (inline style simples)

📁 Estrutura do Código
O formulário está organizado de forma clara e semântica, com:

Uso correto de label associado a cada input

Atributos HTML5 modernos (required, placeholder, min, max, step, etc.)

Agrupamento de opções com optgroup

Estilo básico aplicado ao body e aos label

🎯 Objetivo Educacional
Este exercício faz parte da secção introdutória do curso, focado em:

Compreender os diferentes tipos de inputs HTML5

Aprender a estruturar formulários acessíveis

Praticar organização e boas práticas de marcação

Publicar páginas simples no GitHub Pages
